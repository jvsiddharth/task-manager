
# Task Manager Application

This is a full-stack Task Manager application built as part of the assignment.  
It allows users to create, assign, and manage tasks using a Jira-style Kanban board.

---

## Features

- User signup and login
- Create, update, and delete tasks
- Assign tasks to users
- Task priorities (LOW, MEDIUM, HIGH, URGENT)
- Task status workflow (TODO, IN_PROGRESS, REVIEW, COMPLETED)

---

## Tech Stack

### Backend
- Node.js
- Express
- TypeScript
- Prisma ORM
- PostgreSQL
- Socket.IO

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Axios
